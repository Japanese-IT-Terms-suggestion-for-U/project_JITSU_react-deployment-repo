## はじめに

`勉強したい分野の単語を学習できるようにランダムで出せるサービス`を提供します. <br>
ユーザーの傾向に合った単語を学習することができます。

## クロールするサイト

[분野別用語辞典 ソフトウェア： - IT用語辞典バイナリ](https://www.sophia-it.com/word-category/ソフトウェア)

## 技術スタック

- React
- Laravel

## 機能

1. ログイン・会員登録
2. マイページ
3. 単語カード

## 使用方法

1. ログイン・会員登録を行います。
2. 学習したいテーマを選びます。
3. 単語学習を始めます。
4. 分析できたデータはマイページで確認できます。

## ライセンス

프로젝트 라이센스 정보
